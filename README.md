# Чат‑бот для интернет‑магазина

Консольный чат‑бот на Python с использованием LangChain. Поддерживает:
- ответы на типовые вопросы из `faq.json`;
- проверку статуса заказа по команде `/order <ID>` (данные из `orders.json`);
- диалоговую историю (InMemoryChatMessageHistory);
- логирование сессий в папку `logs/`;
- работу как с **OpenAI API**, так и с **локально развёрнутой LLM** через совместимый API (например, LM Studio, Ollama, LocalAI, vLLM).
